LECTURE 05

HTML Media and Symbol Entities Explanation

Part 1: HTML5 <video> Tag Basics

The <video> tag embeds video in web pages natively without plugins. Example of a simple video tag: 

<video src="my-awesome-video.mp4" controls></video>

Adding the controls attribute enables default browser video controls for play, pause, volume, and fullscreen.

You can also specify width and height: 

<video src="video.mp4" width="640" height="360" controls></video>

The autoplay attribute makes video start automatically but usually requires muted to avoid browser blocking: 

<video src="video.mp4" autoplay muted controls></video>

The loop attribute enables the video to repeat continuously.

The poster attribute sets an image displayed before the video plays: 

<video src="video.mp4" poster="images/video-thumbnail.jpg" controls></video>

Part 2: HTML5 <audio> Tag

The <audio> tag works like the video tag but for sound only.

<p>Listen to our theme song:</p>
<audio src="audio/theme-song.mp3" controls></audio>

Supports attributes like autoplay, loop, and muted.

Part 3: Using Multiple Sources with <source>

To ensure maximum browser compatibility for videos, include multiple source formats inside the <video> tag: 

<video controls poster="images/thumbnail.jpg">
  <source src="videos/my-video.webm" type="video/webm">
  <source src="videos/my-video.mp4" type="video/mp4">
  Sorry, your browser doesn't support embedded videos.
</video>

Part 4: Adding Captions and Subtitles with <track>

The <track> element adds subtitles and captions for accessibility:

<video controls>
  <source src="video.mp4" type="video/mp4">
  <track src="captions_en.vtt" kind="captions" srclang="en" label="English">
  <track src="subtitles_es.vtt" kind="subtitles" srclang="es" label="Español">
</video>

Attributes explained:

  • src: Path to captions (.vtt) file
  • kind: Type of text track (e.g., captions, subtitles)
  • srclang: Language of the track
  • label: Track name shown in player menu

Part 5: Video Format Compatibility

Not all browsers support the same formats. Common formats include:

  • .mp4 (H.264) - widely supported
  • .webm (VP8/VP9) - good support on Chrome, Firefox
  • .ogg (Theora) - older format

Browsers try the <source> list in order and use the first supported format.

Part 6: HTML Symbol Entities

To display special characters safely, use HTML entity names or numbers:

Symbol Description Entity Name Entity Number
© Copyright &copy; &#169;
® Registered Trademark &reg; &#174;
™ Trademark &trade; &#8482;
< Less-than (for showing code) &lt; &#60;
> Greater-than (for showing code) &gt; &#62;
& Ampersand &amp; &#38;
" Double Quote &quot; &#34;
€ Euro Sign &euro; &#8364;

Why use entities for < and >? Because < and > are interpreted as HTML tags. To literally display <p>, write &lt;p&gt;.
