ABOUT HTML...

  • HTML stands for Hyper Text Markup Language
  • It is used to design webpages.
  • HyperText: This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.
  • Markup Language: This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.

TAGS USED IN HTML

  1. The Heading tag
    • Headings are tags used to define titles and subtitles on your page. They are crucial for creating a logical hierarchy and outline for your content.
    • There are 6 heading tags which are used in HTML
    • This is the heading tag which is largest <h1>
    • This heading tag is smaller than h1 <h2>
    • Then this tag <h3>
    • <h4>
    • <h5>
    • The smallest heading tag <h6>

  2. The paragraph tag denoted as "<p>"
    • The paragraph tag is the most common tag you'll use. It's for grouping sentences and blocks of text together.
    • To define a distinct paragraph of text. Browsers automatically add a little bit of space before and after a <p> element, separating it from other content. You should not use multiple line breaks; you should use multiple <p> tags.
  3. The horizontal rule tag: <hr>
    • The <hr> defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.
    • The <hr> element is used to separate content (or define a change) in an HTML page. It is a self-closing tag.
  4. The line break tag: <br>
    • The <br> tag inserts a single line break.
    • It is useful for writing poems or addresses, where the division of lines is significant. It is also a self-closing tag.
  5. The list element tag: <li>
    • The <li> tag defines a list item.
    • It is used inside ordered lists (<ol>) to create a numbered list and inside unordered lists (<ul>) to create a bulleted list.
  6. The anchor tag: <a>
    • The anchor tag is used to create hyperlinks, which are links to other web pages or resources.
    • The href attribute is the most important attribute. It specifies the URL of the page the link goes to. (e.g., <a href="https://www.google.com">Go to Google</a>)
    • Use the target="_blank" attribute to open the linked document in a new browser window or tab.
  7. The image tag: <img>
    • The <img> tag is used to embed an image in a web page. It is a self-closing tag.
    • The src attribute is required and specifies the path (URL) to the image.
    • The alt attribute provides alternative text for the image, which is important for accessibility and is displayed if the image cannot be loaded.
    • You can also use width and height attributes to set the size of the image. (e.g., <img src="my_image.jpg" alt="A descriptive text" width="100" height="100">)
