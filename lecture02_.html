About Table Tag in HTML

HTML Table Tags Overview
Tag Description
<table> Main container for the table. All other table elements go inside this tag.
<tr> Defines a row within the table.
<th> Table header cell. Bold and centered text, used for column headings.
<td> Standard table data cell for content.

Note

Spanning Rows & Columns
  • Use colspan to merge cells horizontally.
    Example: <td colspan="2">This cell spans two columns.</td>
  • Use rowspan to merge cells vertically.
    Example: <td rowspan="3">This cell spans three rows.</td>

Nested Lists

The key rule for creating a nested list is that the child (inner) list must be inside an <li> element of the parent (outer) list.

Steps:
1. Start your main list (<ol> or <ul>).
2. Create a parent list item (<li>).
3. Inside the parent <li>, create a new (nested) list.
4. Add sub-list items.
5. Close the inner list and then the parent <li>.
